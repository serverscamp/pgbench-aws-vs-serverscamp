# Runbook: bench10000

Disk-bound workload: scale 10000 (~150 GB), warmup 10 min, three 20-min measurement runs.
Same shape as `bench100`/`bench1000`, only the durations and DB name change.

## Environment

```bash
export PGHOST="<db-endpoint>"
export PGPORT="5432"
export PGUSER="<db-user>"
export PGPASSWORD='<password>'
export PGSSLMODE="require"
```

## Script

```bash
settle() {
  echo "[$(date +%H:%M:%S)] CHECKPOINT + waiting for autovacuum..."
  psql -d bench10000 -c "CHECKPOINT;" >/dev/null
  while [ "$(psql -d bench10000 -tAc "SELECT count(*) FROM pg_stat_activity WHERE backend_type='autovacuum worker'")" != "0" ]; do
    echo "  autovacuum still working..."
    sleep 10
  done
  echo "[$(date +%H:%M:%S)] ready"
}

mkdir -p ~/results/bench10000 && cd ~/results/bench10000

settle
echo "=== WARMUP ==="
pgbench -d bench10000 -c 32 -j 4 -T 600 -M prepared --no-vacuum

for i in 1 2 3; do
  settle
  echo "=== RUN $i ==="
  pgbench -d bench10000 -c 32 -j 4 -T 1200 -M prepared --no-vacuum \
    --log --log-prefix=run${i} --aggregate-interval=10 \
    2>&1 | tee run${i}.txt
done

echo "=== DONE bench10000 ==="
```

## Parameters

| Flag | Value |
|---|---|
| `-c 32 -j 4` | 32 clients, 4 worker threads |
| `-M prepared` | prepared statements |
| `-T 600` | warmup 10 min |
| `-T 1200` | measurement 20 min |
| `--no-vacuum` | skip pgbench's pre-test VACUUM |
| `--log --log-prefix=runN --aggregate-interval=10` | per-thread aggregate logs every 10 s |

Repeats: 3 measurement runs.
